# MessMate 🍽️

Smart Hostel Mess Management System

A full-stack web application for managing hostel mess operations.

## Features
- User authentication (JWT)
- Meal selection (Breakfast, Lunch, Dinner)
- Monthly meal subscription system
- Daily override functionality
- Automatic bill calculation
- Admin dashboard (view student selections)

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas

## How to Run
1. Clone repo
2. Run `npm install` in backend & frontend
3. Add `.env` file
4. Run `npm run dev`
